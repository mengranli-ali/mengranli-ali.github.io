### Array 序列

Definition:

指它的成员都有序排列，并可以通过下标偏移量访问到它的一个或几个成员.

Array includes 字符串 `String `，列表 `List` ，元组 `Tup`.

先定义一个字符串来保存生肖 - create a variable:

```vim

print(chinese_zodiac[0:4])
```
